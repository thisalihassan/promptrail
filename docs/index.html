Features

Trace every AI change

Everything you need to understand, navigate, and undo AI edits.

Prompt Timeline

Every AI prompt listed chronologically with file change counts, model badges, and timestamps. See exactly which files each prompt touched.

promptrail timeline --files -n 10

Cherry-Pick Rollback

Undo one prompt's changes without losing everything after it. Surgical precision, not a full revert.

promptrail rollback 5

Prompt-Scoped Diffs

Before/after diffs scoped to a single prompt, not the entire session. See exactly what changed and why.

promptrail diff 3

Search Everything

Full-text search across all prompts, AI responses, and file names. Find which prompt edited a specific file or mentioned a topic.

promptrail search "database optimization"
How It Works

No interception. No config.

Promptrail reads session data already on disk. Nothing to install or configure.

01

Use AI normally

Write prompts in Cursor, Claude Code, or VS Code Chat as you always do. Change nothing.

02

Auto-detected

Promptrail reads the session data these tools already write to disk. Zero runtime overhead.

03

Browse, diff, rollback

CLI, VS Code sidebar extension, or Claude Code plugin. Your choice of interface.
