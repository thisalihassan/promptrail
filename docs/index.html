The Problem

Your AI agent edited 47 files.
Do you know why?

Without Promptrail

You ask Cursor to refactor auth. It edits 4 files.
You ask it to add error handling. 6 more files.
Something breaks.

Git log shows one commit: "AI changes"
Cursor checkpoints: a flat list of file states.

Which prompt caused which change? No idea.

With Promptrail

$ promptrail timeline --files

● #0 cursor Refactor auth middleware  4 files
      src/middleware/auth.ts
      src/routes/login.ts
      src/routes/logout.ts
      src/utils/token.ts
● #1 cursor Add error handling  6 files
      src/middleware/error.ts
      src/routes/api.ts
      ...
○ #2 cursor Why is the build failing?  no changes
Features

Everything you need to trace AI changes

☰

Prompt Timeline

Every AI prompt listed chronologically with file change counts, model badges, and timestamps. Filter by source, model, or limit with -n 10. Each prompt shows exactly which files it touched.

↺

Cherry-Pick Rollback

Undo one prompt's changes without losing everything after it. promptrail rollback 5

⇄

Prompt-Scoped Diffs

Before/after diffs scoped to a single prompt, not the whole session. promptrail diff 3

🔎

Search Everything

Full-text search across prompts, AI responses, and file names. Find which prompt edited auth.ts or mentioned "database optimization". Results show #N so you can immediately diff or rollback.

How It Works

No interception. No configuration.

1

2

3

CLI, extension sidebar, or Claude Code plugin. Your choice.
